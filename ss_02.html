SS02-01

Display Trading License Information

This services displays the details of a Trading License & the related details such as (Undertaking, Violations, and Work Permits)

1- Login as "Inspector" "المفتش"

1- The User clicks on "مشاهدة ملف المنشأة" button for the selected Establishment in the list of "البحث - الرخص التجارية" page.

2- The System displays the "Establishment Profile/معلومات الرخصة التجارية" and fetches all related data from SAP Trading License System & from DARB Back-End System:

    2.1- Undertakings - التعهدات: The System gets all Undertaking records related to the displyed Trading License from SAP Trading License Back-End System, through an Integration Point.

 

    2.2- Violations - المخالفات: The System gets all Violations records related to the displyed Trading License from DARB Back-End System.

 

    2.3- Work Permits - التصاريح: The System gets all Permit records related to the displyed Trading License from DARB Back-End System.

1. Integration Point with SAP Trading License System is required.
