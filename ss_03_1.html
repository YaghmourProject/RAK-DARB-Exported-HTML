SS0301

Display Regions List

1-Login as "مدير النظام"

1- click on "إدارة المناطق"  in main menu on the right side

2- system shall display paged list of all regions defined in the system, ordered alphabitically

SS0302

Search for specific Regions as per specific criteria

1-Login as "مدير النظام"

 

1- click on "إدارة المناطق"  in main menu on the right side

2- system shall display paged list of all Regions defined in the system, ordered alphabitically

3- fill any of listed search criteria and click search button

4- system shall display filtered paged list of all Regions defined in the system, ordered alphabitically, as per filled search criteria

SS0303

Sort search result using any of listed columns

1-Login as "مدير النظام"

 

1- click on "إدارة المناطق"  in main menu on the right side

2- system shall display paged list of all regions defined in the system, ordered alphabitically

3- user may click any of displayed columns headers, to order the list using that column ASC and DESC

SS0304

Update Region

1-Login as "مدير النظام"

2- display list of Regions

1- click on "إدارة المناطق"  in main menu on the right side

2- system shall display paged list of all Regions defined in the system, ordered alphabitically

3- user click "تعديل" button beside any of displayed list of Regions

4- system display screen to allow user to edit region details 

5- system display success message that the region updated successfully

SS0305

Delete Region

1-Login as "مدير النظام"

2- display list of Regions

1- click on "إدارة المناطق"  in main menu on the right side

2- system shall display paged list of all Regions defined in the system, ordered alphabitically

3- user click "حذف" button beside any of displayed list of Regions

4- system display screen to allow user to select another region from list of defined regions, to merge all missions related to this region

5- system display success message that the region deleted successfully, and all related missions merged successfully to region X
