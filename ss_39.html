SS3901

Display Seizured Goods details List

1-Login as "امين المستودع"

1- click on "إدارة المحجوزات"  in main menu on the right side, then click "إدارة حجز البضائع"

2- user shall click "عرض تفاصيل البضاعة"  beside any of listed seizured goods

3- system shall display paged list of all Seizured Goods details defined in the system, ordered alphabitically

SS3903

Sort using any of listed columns

1-Login as "امين المستودع"

 

1- click on "إدارة المحجوزات "  in main menu on the right side, then click "إدارة حجز البضائع"

2- system shall display paged list of all Seizured Goods defined in the system, ordered alphabitically

3- user shall click "عرض تفاصيل البضاعة"  beside any of listed seizured goods

4- user may click any of displayed columns headers, to order the list using that column ASC and DESC

SS3904

Add selected Seizured Goods To the Store

1-Login as "امين المستودع"

2- display list of Seizured Goods

1- click on "إدارة المحجوزات "  in main menu on the right side, then click "إدارة حجز البضائع"

2- system shall display paged list of all Seizured Goods defined in the system, ordered alphabitically

3- user shall click "عرض تفاصيل البضاعة"  beside any of listed seizured goods

4- user shall select/check one or more of listed goods to be added to the store

5- user click "اضف الكل للمستودع" button beside any of displayed list of Seizured Goods 

6- system display success message that all selected Seizured Goods added to the store successfully
