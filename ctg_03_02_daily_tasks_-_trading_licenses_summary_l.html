Integration

Description

This service allows the user to view counters summary list for Trading Licenses to be inspected in daily bases, once the User clicks on the 'Daily Tasks / مهمات يومية' Tab Page. These daily tasks are generated by the Integration Back-end Systems, assigned to each inspector by their Supervisor. The System only retrieves the Tasks for the Current System Date, Postponsed Tasks (future tasks dates) are not displayed here.
