SS4205

Add New Work Permit - Inside RAK City

This service allows the User to submit a new request for Work Permits (on behalf of the Investor).

1-Login as "موظف الإستقبال"

 

1- The User clicks on "إدارة التصاريح" in the side menu.

2- The System displays "قائمة التصاريح" page.

3- The User clicks on "طلب تصريح جديد" button.

4- The System redirects the User to the "طلب تصريح" page.

5- The User selects the Permit Type "تصريح لرخصة من داخل رأس الخيمة".

6- The System displays the required entry fields for the selected Permit Type.

7- The User enters the Trading License Number and clicks on "بحث" button.

8- The System searches for the entered Trading License Number in SAP Tradin License System:

    8.1- Found (with No Violations): the System retrieves and displays the related information of the selected Trading License Number.

    8.2- Found (with Violations): the User instructs the client to go first to pay all registered violations (at the Trading License Department) before requesting a new Permit.

    8.3- Not Found: the System displays a friendly message to the User that no record found for the select Trading License Number, and the User  instructs the client to go first and proceed with requesting a New Trading License Record (at the Trading License Department) before requesting a new Permit.

9- The User clicks on "متابعة" button to proceed with the next step Work Permit Details.

10- The User selects "فئة التصريح" from the Dropdown list (تجاري/مهني).

11- The System populates "نوع التصريح الرئيسي" dropdown list based on the selected point in Step #10.

12- The User selects the requested "نوع التصريح الرئيسي" from the dropdown list.

13- The System displays the entry form fields related to the selected valued in the "نوع التصريح الرئيسي" dropdown list.

14- The User fills the required form fields and clicks the "تقديم" button.

15- The System saves the Permit's data and displays the Reference Number for the submitted request.

16- The System sends the following Notifications:

     16.1- Email and SMS Notifications to the client containing the Reference Number for the submitted request and the status is Under Processing.

     16.2- Email Notification to the Department Manager to review the submitted Work Permit Request, and to take the required action (Approve or Reject) the Request.

17- Upon Manager Approval (in the Next Step); The System generates two inspection tasks and assigns them to the Inspector who is mapped to the Establishment Geo-Location:

     17.1- First Inspection Task: on the Work Permit Start Date.

     17.2- Second Inspection Task: on the Work Permit End Date.
