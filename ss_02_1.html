SS0201

Display ispections missions List

1-Login as "مسئول التفتيش"

1- click on "إدارة المفتشين"  in main menu on the right side

2- system shall display paged list of all inspectors defined in the system

3- user shall click on button "مهام المفتشين"

4- system shall display paged list of all inspection missions, ordered DESC by inspection date

SS0202

Search for specific inspection mission as per specific criteria

1-Login as "مسئول التفتيش"

 

1- click on "إدارة المفتشين"  in main menu on the right side

2- system shall display paged list of all inspectors defined in the system

3- user shall click on button "مهام المفتشين"

4- system shall display paged list of all inspection missions, ordered DESC by inspection date

5- fill any of listed search criteria and click search button

6- system shall display filtered paged list of all inspecion missions defined in the system,ordered DESC by inspection date, as per filled search criteria

SS0203

Sort search result using any of listed columns

1-Login as "مسئول التفتيش"

 

1- click on "إدارة المفتشين"  in main menu on the right side

2- system shall display paged list of all inspectors defined in the system

3- user shall click on button "مهام المفتشين"

4- system shall display paged list of all inspection missions, ordered DESC by inspection date

3- user may click any of displayed columns headers, to order the list using that column ASC and DESC

SS0204

Assign Inspector

1-Login as "مسئول التفتيش"

2- display list of inspection missions

3- user shall click on button "مهام المفتشين"
